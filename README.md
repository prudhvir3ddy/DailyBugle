## local setup instructions 
Go to <http://newsapi.org/> and generate your API key there and 
append to your local.properties like this

``` groovy
sdk.dir=YOUR_LOCAL_PATH_TO_ANDROID_SDK
api.news=ADD_YOUR_API_KEY_HERE

```
